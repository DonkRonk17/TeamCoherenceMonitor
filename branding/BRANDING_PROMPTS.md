## 1. TITLE CARD (16:9 Landscape - 1920x1080px)

**For:** GitHub repository header image, presentations, documentation

**DALL-E Prompt:**
```
Create a sleek, modern title card for "TeamCoherenceMonitor" - a real-time coordination health dashboard for multi-agent AI teams.

STYLE:
- Dark navy blue background (#0a0e27)
- Bright cyan accents (#00d4ff)
- Tech-forward, professional aesthetic
- Clean, minimalist design

LAYOUT:
- Large bold title "TeamCoherenceMonitor" centered
- Smaller tagline below: "Real-Time Coordination Health Dashboard"
- Central visual element: A circular gauge/meter showing "85" with a glowing cyan arc, representing the coherence score
- Below the gauge: 5-7 small connected nodes representing team agents, each with a subtle glow
- Subtle connection lines between nodes suggesting coordination
- Background has very subtle circuit patterns and data flow visualizations

VISUAL ELEMENTS:
- The coherence score gauge should be prominent and easy to read
- Agent nodes should be arranged in a semi-circle or horizontal line
- Use glowing effects for the cyan accents
- Include small status indicators (dots) near some nodes

MOOD: Professional, mission-critical, real-time monitoring, team coordination, trust
```

## 2. LOGO MARK (1:1 Square - 1024x1024px)

**For:** Profile pictures, app icon base, social media avatars

**DALL-E Prompt:**
```
Create a minimalist logo icon for "TeamCoherenceMonitor" - a team coordination health monitoring tool.

STYLE:
- Simple geometric design
- Bright cyan (#00d4ff) on dark navy (#0a0e27)
- Clean, recognizable at small sizes
- Professional and modern

DESIGN:
- Central element: A circular gauge or meter showing approximately 3/4 filled
- Inside or around the gauge: 5 small dots arranged in a pentagon pattern, representing team members
- The dots are connected by thin lines forming a network shape
- One prominent central dot or node
- No text

CONCEPT:
- The circular gauge represents the coherence score
- The connected nodes represent team coordination
- The overall shape should form a cohesive, unified icon

CONSTRAINTS:
- Must be readable at 32x32px
- Icon only, no text
- Use only 2-3 colors maximum
- Clean silhouette that works on any background

MOOD: Monitoring, health check, team unity, coordination, professional
```

## Usage Notes

### Generation Instructions

1. **Use these prompts with DALL-E 3** (ChatGPT or OpenAI API)
2. **Generate multiple variations** and select the best
3. **Maintain consistency** across all assets
4. **Resize carefully** - maintain aspect ratios

### File Naming Convention

After generation, save files as:
- `title_card_teamcoherencemonitor.png` (1920x1080)
- `logo_mark_teamcoherencemonitor.png` (1024x1024)
- `horizontal_logo_teamcoherencemonitor.png` (1800x600)
- `app_icon_teamcoherencemonitor.png` (512x512)

### Quality Checklist

- [ ] Title card is 16:9 aspect ratio
- [ ] Logo mark works at 32x32px
- [ ] Horizontal logo is readable at 50% size
- [ ] App icon passes the "squint test" (recognizable when blurry)
- [ ] All assets use consistent colors
- [ ] No text in logo mark or app icon
